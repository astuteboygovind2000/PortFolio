MedFinder Project Overview

This project is a web-based platform designed for coaching centers or educational institutions to manage student fees, track dues, and handle installment payments efficiently. The system provides role-based access to three different types of users: Admin, Accountant, and Student, each with distinct permissions and functionalities.

Key Features:

  1. Admin Dashboard:
    • Full access to all system functionalities.
    • Can view and manage student profiles, fee structures, payment histories, and installments.
    • Update courses in coaching and manage ( edit and delete ) that courses.
    • Ability to add, update, or delete student information.
  2. Accountant Interface:
    • Focused on financial management.
    • Register any course on student profile.
    • Can update payment statuses, manage fee collections, and record installment plans for students.
    • Track outstanding dues and send payment reminders to students.
    • View and generate fee-related reports.
  3. Student Portal:
    • Personalized access for students to view their fee details.
    • Can track outstanding dues, view installment plans, and payment history.
    • Can see the old and new joined courses.

Technology Stack:

  • Frontend:HTML, CSS, JS, Bootstrap
  • Backend:Django Python frameword
